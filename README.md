101429091_comp3123_labtest2

A simple and responsive Weather App built with React and Vite.
Users can search for any city and view real-time weather information such as temperature, conditions, humidity, wind speed, pressure, and more.

This project was completed for COMP3123 – Lab Test 2.

Live Demo (Vercel Deployment):https://vercel.com/hamza-hafezs-projects/101429091-comp3123-labtest2/89LN48n6gMxxfYMT3ygRuwPS4MCY

Features

-Search weather by city name

Displays:

  - Current temperature (°C)

  - Feels-like temperature

  - Weather condition & icon

  - Humidity

  - Pressure

  - Wind speed

  - Daily min/max temperature

-Modern, clean UI with a glassmorphism-inspired design

-Fully responsive for both desktop and mobile

Tech Stack:

-React (Vite) – front-end framework
-Axios – HTTP client for API requests

-OpenWeatherMap – weather data source

-CSS – custom UI styling
